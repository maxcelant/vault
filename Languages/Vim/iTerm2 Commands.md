| Command                    | Motion                   |     |
| -------------------------- | ------------------------ | --- |
| `CMD + D`                  | Divide terminal into two |     |
| `CMD + Option + Arrow Key` | Switch between panes     |     |
| `CMD + W`                  | Close a pane             |     |
| `CMD + SHIFT + D`          | Vertical Pane            |     |
|                            |                          |     |
