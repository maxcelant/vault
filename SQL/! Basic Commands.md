##### Creating a database
`createdb <db_name>`

##### Connecting to database
`psql -d <db_name>`

##### All tables
`\d`

##### Running a Script
`psql -d <db_name> -f <script_file>`

##### Look at table
`\d <table_name>`

##### Inline Commands
`<query_statement>;`

##### End of Query
- `q` skips to end
- `z` goes down
